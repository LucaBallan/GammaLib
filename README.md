# GammaLib
Generic Computer Vision Library

Generic Computer Vision Library supporting
 - Kinematic chains/trees
 - 2D/3D deformable meshes
 - Camera projection models
 - Kd-Tree
 - OcTree
 - QuadTree
 - BSP Tree
 - Wavelet
 - Tensor operations
 - OpenGL Rendering
 - OpenGL Shaders (GLSL)
 - Media player for win32
 - Video input
